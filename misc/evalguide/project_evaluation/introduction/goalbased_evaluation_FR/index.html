Evaluation basée sur les objectifs

Avec l'évaluation basée sur les objectifs il s'agit de déterminer si le projet a atteint ses objectifs. Cette question se pose fréquemment à la fin du processus du projet dans le cadre d'une évaluation sommative .

évaluation sommative

"L’évaluation sommative sert à contrôler la qualité, les effets et l’utilisation d’une offre de formation. On s’intéresse à la question de savoir si une offre de formation ou certaines de ses composantes répondent à certaines attentes lors de l’utilisation pratique." (traduit de Tergan, 2000)

Etant donné qu’il y a autant d'objectifs que de projets, ce manuel ne peut pas fournir d’instruments de collecte concrets pour l'évaluation basée sur les objectifs. Nous ne proposons donc que des lignes directrices générales :

1. Formulez des objectifs clairs pour votre projet

Il faut que les objectifs aient été clairement formulés pour que l’on puisse vérifier s’ils ont été atteints. Des indications sur la formulation d'objectifs se trouvent dans le chapitre 2. Choisir les instruments, au niveau du concept, à la page objectifs.

2. Faites le lien avec les objectifs du programme

Si votre projet est soutenu dans le cadre d'un programme plus général, réfléchissez à quel objectif du programme il contribue. Les objectifs de votre projet devraient contribuer en tant qu’"objectifs intermédiaires" à la réalisation des objectifs généraux formulés au niveau du programme (voir la page hiérarchies d'objectifs).

3. Comment voulez-vous atteindre votre objectif?

Réfléchissez pour savoir quelles mesures dans votre projet contribuent à la réalisation de quels objectifs (voir la page modèles logiques).

4. A quel indice reconnaîtrez-vous si l'objectif a été atteint ?

Réfléchissez pour identifier les indicateurs observables ou mesurables  permettant d’apprécier si les objectifs ont été atteints.

5. Comment allez vous mettre à disposition les données nécessaires ?

Réfléchissez à la méthode appropriée pour mettre à disposition les informations nécessaires. Plus d'informations sur les méthodes d'évaluation se trouvent dans la partie générale sous : méthodes d'évaluation.

Pour la mise en œuvre d'une évaluation basée sur les objectifs, référez-vous au processus d'évaluation proposé dans ce manuel (1. planifier l'évaluation, 2. choisir les instruments, 3. collecter les données, 4. évaluer les données et 5. rédiger le rapport).
