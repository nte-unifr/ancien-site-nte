Content analysis

Content-analytical methods are available for the analysis of data of an interview, a non-structured observation or of a questionnaire with open questions. Content analysis makes a distinction between quantitative and qualitative approaches.

Quantitative content analysis

The aim of a quantitative content analysis is to reduce the complexity of the data material and transform it into a statistically analysable form, for which the procedure is as follows:

  1. Firstly, a theory-based category system is created which covers all of the aspects relevant to answering the question. If, for example, the cognitive learning strategies of the students who work with a learning program is to be investigated, the category system to be created should include learning strategies of interest (e.g. repetition, elaboration).
  2. The data material (e.g. protocols of “thinking-aloud”) is broken down into units of meaning - these can be one or two sentences but also only one word.
  3. Individual units of meaning (propositions) are then allocated to individual categories. This usually occurs by assigning each unit of meaning to a a code that represents a certain category.

The suitability of the category system is then usually tested using two or three sample texts  before beginning with the actual coding. An example for a category system that also includes example propositions can be found here: Category system for learning strategies (pdf, 32 kB, in German)

With the help of such analyses, conclusions can be drawn as to whether certain learning strategies occur more than the others, or (if a conclusive learning test is employed) whether those students who utilise certain learning strategies learn more successfully.


Qualitative content analysis

A category system is also created in qualitative content analysis - however, the categories often arise from the text material, therefore they are not defined in advance. The aim is not a quantification but an interpretation of text content, i.e. the contentual meaning of statements is identified without reducing the material to quantifiable statements. Such approach allows three different aims to be pursued:
Summary, explication and structuring. Summary aims to reduce the interview material. However, the essential content should be preserved in order to abstract a reproduction of the original material. Explication involves adding additional material to ambiguous text parts. This completes and annotates the relevant passages of the text. Structuring aims to filter out a particular structure from the material.

The approach of the qualitative content analysis to be undertaken differs depending on the aim to be pursued. Further information about qualitative content analysis can be found at Mayring (2000) at: http://www.qualitative-research.net/fqs-texte/2-00/2-00mayring-e.htm
