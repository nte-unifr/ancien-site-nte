Analyser les données

Selon la méthode de recueil des données et la question d'évaluation, les données peuvent être analysées au moyen d'une multitude de procédures d'analyse statistique ou de contenu. Dans ce manuel, nous ne pouvons pas présenter en détail toutes les procédures d'analyse, nous nous limitons ainsi à des informations de base et à mettre à disposition, pour certaines questions d'évaluation, des exemples d'évaluation.


Analyse de données d'un catalogue de critères

L’analyse des données issues d'un catalogue de critères est déterminée par le mode d’analyse défini dans le cadre du catalogue lui-même. Il convient de respecter celle-ci pour éviter des altérations.

Évaluation des données d'une analyse de réseau

Les différents projets au sein d’un programme représentent les nœuds du réseau dont on veut montrer les interrelations.

  • Exemple d'évaluation

  • Une possibilité est de représenter les relations entre les projets constatées avec un questionnaire d'analyse du réseau, simplement comme des relations entre nœuds (cf. Netzwerkdarstellung 1 (pdf, 276 kB, en allemand) ; une représentation pour chacune des dimensions : orientation, connaissance, etc.)
    Les dimensions orientation / connaissance et entrée / sortie peuvent en outre être visualisées dans un système de coordonnées : Netzwerkdarstellung 2 (pdf, 26 kB, en allemand).
    D'autres possibilités d'évaluation sont fournies par les outils d'analyse de réseau, p. ex. UCINET (http://www.analytictech.com/ucinet/ucinet.htm).

Procédures d'analyse statistique

Vous pouvez utiliser une procédure d'analyse statistique si vous avez recueilli des données quantitatives (numériques) par questionnaire, entretien, test, observation, etc. Ces procédures statistiques peuvent servir à décrire le matériel de données, p. ex. par des mots clés, ou sous forme graphique (statistique descriptive). Elles peuvent aussi servir à vérifier des hypothèses de différence (statistique inférentielle).
