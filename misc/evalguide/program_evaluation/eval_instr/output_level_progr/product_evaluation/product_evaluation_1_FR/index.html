Résumé des résultats de l'évaluation de projets

Les niveaux d'évaluation présentés dans le chapitre sur l'évaluation de projets contiennent des dimensions d'évaluation génériques qui sont pertinentes pour beaucoup de projets elearning.

Pour l'évaluation de programmes, il est donc profitable de condenser les données rassemblées au niveau des projets pour ces dimensions, comme illustré dans le tableau suivant :


program_result_eval_EN



Les projets a et b représentent à titre d'exemple une quantité de projets qui ont utilisé un instrument donné (I.) pour recolter des données sur une dimension d'évaluation donnée (p. ex. satisfaction des étudiants par rapport à l'offre de formation).

Les résultats (R.) obtenus au niveau des projets d'une même dimension d'évaluation peuvent être agrégés au-delà des projets individuels. Par exemple pour faire des déclarations concernant la satisfaction globale des étudiants par rapport aux offres de formation développés dans le cadre du programme.

  • Si les mêmes instruments de collecte ont été utilisés pour les évaluations des projets  (p. ex. le même questionnaire de satisfaction), et pour autant que cela semble judicieux, on peut p. ex. calculer, à partir des moyennes pour chaque projet, une moyenne globale pour tous les projets.

  • Si des instruments différents ont été utilisés dans les projets pour répondre à la même question d'évaluation, ou s'il ne semble pas judicieux de calculer une moyenne pour tous les projets, les résultats peuvent également être résumés du point de vue du contenu et / ou utilisés pour comparer les différents projets.

  • Le résumé des résultats réussit d'autant plus facilement que les différents projets ainsi que les instruments utilisés dans l'évaluation des projets se ressemblent. (Aux niveaux d'évaluation inférieurs, cela est plus facilement réalisable qu'aux niveaux plus élevés : trouver un questionnaire sur l’utilisabilité applicable à tous les projets est plus simple que de déterminer un instrument pour l'analyse du transfert qui soit pertinent pour tous les projets !)
