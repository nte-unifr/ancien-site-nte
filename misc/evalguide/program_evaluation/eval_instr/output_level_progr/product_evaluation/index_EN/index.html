Product assessment

There are two options for assessing the products of the individual projects:

1. Use of data from project evaluations for program evaluation. Preferably, tThis procedure requires preferably all individual projects to provide certain data (e.g. those forof usability or satisfaction) from their evaluation (check see also the different tools in the chapter on Project evaluation.)

  • Further information about the compilation of outputs from project evaluations

2. Collection of evaluation data especially for program evaluation. If no project evaluations have been performed or if structured data is not available from project evaluations, it is necessary to collect data specifically for program evaluation.

  • The products available can be assessed, for example by experts, using a criteria catalogue.
  • The students (as the most important target group of projects) can be asked to give feedback on the products using a generic assessment questionnaire (doc, 65 kB, in German).
