Machbarkeitsstudie

Verschiedene Fragen können im Mittelpunkt von Machbarkeitsstudien stehen (vgl. Projekt Magazin, 2005):

  • Ist das Projekt technisch machbar?
  • Ist es finanziell machbar?
  • Ist es im abgesteckten Zeitrahmen machbar?
  • Mit welchen Risiken ist das Projekt verbunden?

Bei der Durchführung einer Machbarkeitsstudie gibt es keine festgelegten Methoden. Je nach Projekt oder Programm, welches auf die Machbarkeit überprüft werden soll, können verschiedene Methoden angewendet werden. Beispiele hierfür sind:

  • Literaturrecherche (wissenschaftliche Literatur, Projektberichte etc.): Aufzeigen von ähnlichen, bereits verwirklichten Projekten (positiver Machbarkeitsnachweis) bzw. von Widersprüchen zwischen den Projektzielen und vorhandenen Erkenntnissen (negativer Machbarkeitsbeweis)
  • Computergestützte Simulationen
  • Expertenbefragungen bzw. Interviews

Das Ergebnis einer Machbarkeitsstudie ist ein Bericht, der die folgenden Themen beinhalten kann (vgl. Jenny, 2001):

  1. Ausgangslage
  2. Ziele und Lösungsansätze
  3. Wirtschaftlichkeitsanalyse
  4. Risikoanalyse
  5. Projektorganisation
  6. Projektgesamtplanung
  7. Aufgliederung in Teilprojekte

Eine Anleitung zur Durchführung von Machbarkeitsstudien können Sie als pdf-Dokument von folgender Site herunterladen: http://www.oleimeulen.info/content/references.htm (dort unter 'Machbarkeitsstudien')
