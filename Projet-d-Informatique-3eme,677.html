Projet d’Informatique 3ème semestre (2006),  17 septembre 2006

Année universitaire 2006-2007. Cours destiné aux étudiants en informatique de bachelor.

Gérald Collaud  Jacques Monnard 

Année universitaire 2006-2007

Exigences préalables

Avoir suivi les cours Programmation I (Programmation fonctionnelle), Programmation II (Programmation impérative), Programmation IIA (Paradigmes de programmation) et avoir fait le projet d’informatique de première année.

Encadrement
 Gérald Collaud,
 Jacques Monnard.

Informations générales
 Ampleur : 5 ECTS
 Le cours a lieu le mercredi de 8h15 à 10h00. La salle de cours doit encore être déterminée.
 Première séance : mercredi 25 octobre 2006
 Langue : Français
 Heures de réception : seront indiquées au cours

Site du cours

http://nte.unifr.ch/moodle/course/view.php?id=886

Description

Le but du projet est de se familiariser avec le développement d’applications web dynamiques liées à des bases de données.
Dans le cadre de ce projet, les étudiants développeront une application (par exemple un "content management system" (CMS)) avec le langage de programmation php et la base de données mySQL.

Bibliographie
 PHP and MySQL Web Development, Luke Welling, Laura Thomson, SAMS, 2003
 Web Database Applications with PHP & MySQL, Hugh E. Williams, David Lane, O’Reilly, 2002
 PHP MySQL Website Programming : Problem - Design - Solution, Chris Lea, et al, APress, 2003
