A Java version of gLPL

This applet lets you build your own gLPL model. Note that there are a two main differences between this version and the original gLPL language definition :

  • indices can not be included and
  • the type of the restriction is defined by the name of the RHS.

You can see here a more complete example of a gLPL model.

When you have your final LPL code, you can copy and paste it in the on-line solutions system to get the optimal solution.
